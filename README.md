# TASKMANAGER
DESCRIPCION
Es un sistema diseñado para organizar, asignar, supervisar y dar seguimiento a las tareas y actividades de un equipo de trabajo. Su propósito es mejorar la coordinación, colaboración y productividad de los integrantes, permitiendo gestionar proyectos de manera eficiente, controlar avances, optimizar recursos y asegurar el cumplimiento de los objetivos y plazos establecidos.
